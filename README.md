# Miles to Kilometers Converter 

A simple Python GUI application built using **Tkinter** that converts miles to kilometers.

## What the program does
- Allows the user to enter a distance in miles
- Converts the value to kilometers when the **Calculate** button is clicked
- Displays the result instantly in the GUI
